<decisions>
## Implementation Decisions

### Integration Pattern
- **Zelfstandige State Machines**: Elk theorieconcept wordt geprogrammeerd als zijn eigen onafhankelijke state machine (bijv. `PandoraTracker`, `ReverseLevelTracker`, `TrendsDetector`). Ze koppelen in op de gedeelde CCTA theorie resultaten, maar behouden hun eigen gecapsuleerde status (State). Voldoet strict aan het Single Responsibility principe.
- **Geen Decorators**: Module klassen "plakken" de methodes niet onofficieel op de bestaande klasse zoals een OriginTracker.

### Agent's Discretion
- **Meting parameters Deep Dives / Pandora / Trends**: Agent discretion mag logische drempels voor piercing wicks, invalidatie targets en "neutral zone" berekeningen toepassen op basis van theoriemodellen.
</decisions>
